Github_Tutorial_Beginner

Github tutorial for working with github for the first time.

Let's Start!

  1. First go to Github.com and sign up for GitHub or sign in if you already have an account.
  2. Click on the ( in the repository menu in the top left.
  3. Now Create the repository by using the following settings:
  4. Repository name = Github_Tutorial_Beginner
  5. Description = Github tutorial for working with github for the first time.
  6. Set to Private (so it is only viewable for you, this is changed later)
  7. Select “Add a README file”
  8. Press the “Create repository” button

Now your almost empty repository is made. The only information inside is the README.md file. This file contains the main information about your study/review/game/tool/or whatever you want in your github. Now it looks something like this:

  1. Now click on the :pencil2: icon to add the README file.
  2. To line 3 add the following: Whoohoo, I'm working in Github for the first time! :grin:
    At the bottom in the Commit changes section add the discription: Added my first line!
    Now use the Commit changes to change the file.

These descriptions can help to remember what you've done at a later stage.

Now we can see that our senteance isn't separated by an enter. So we're going to change the file again!

  1. To change this we can add an extra enter, so line 3 becomes an empty line or we can place a <'br> (without ') at the end of the senteance.
    For now we will use the extra enter on line 3
