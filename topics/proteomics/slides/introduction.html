FASTA File Format - Text based file to store DNA, RNA, protein sequences in a single letter code - Each entry has - One header line: starting with `>` and containing a unique identifier - A (nucleotide/amino acid) sequence ![]({{site.baseurl}}/topics/proteomics/images/intro-slides/fasta-proteomics.png) --- # FASTA Files for Proteomics - A proteome is the set of proteins thought to be expressed by an organism. - Only proteins that are present in the FASTA file can be identified - The more proteins are present in the FASTA file the higher the changes for false identifications - Choose the right FASTA file - Source for proteome FASTA files: - Uniprot, neXtProt, NCBI, own/public DNA or RNA sequencing data .footnote[ Tutorial: [Protein FASTA database handling](https://training.galaxyproject.org/training-material/topics/proteomics/tutorials/database-handling/tutorial.html) ] --- #
